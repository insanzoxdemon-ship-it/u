Endpoints & Documentation
POST/api/proxy/package/{slug}/days/{days}/user/{username}/pass/{password}

Creates a new licensed user in the specified package.

POST https://authcs.vercel.app/api/proxy/package/aimkill/days/30/user/john/pass/pass123
X-API-KEY: YOUR_KEY
POST/api/proxy/user/{username}/resethwid

Clears the HWID lock for a user.

POST https://authcs.vercel.app/api/proxy/user/john/resethwid
X-API-KEY: YOUR_KEY
DELETE/api/proxy/user/{username}/delete

Permanently deletes a user. Irreversible.

DELETE https://authcs.vercel.app/api/proxy/user/john/delete
X-API-KEY: YOUR_KEY
GET/api/proxy?appid={slug}&type=fetchallusers

Fetches all users for a given package.

GET https://authcs.vercel.app/api/proxy?appid=insanzocheatsaimkill&type=fetchallusers
X-API-KEY: YOUR_KEY
