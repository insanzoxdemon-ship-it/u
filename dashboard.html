Endpoints & Documentation
POST/api/proxy/package/{slug}/days/{days}/user/{username}/pass/{password}

Creates a new licensed user in the specified Auth.cs application. Use -1 for lifetime. Requires X-API-KEY header.

POST  https://authcs.vercel.app/api/proxy/package/aimkill/days/30/user/john_doe/pass/secret123
Headers: X-API-KEY: YOUR_API_KEY_HERE  Content-Type: application/json
Response: { "success":true, "message":"User created successfully", "username":"john_doe", "package":"aimkill", "expires":"30 days" }
POST/api/proxy/user/{username}/resethwid

Clears the HWID lock for a user, allowing authentication from a new device. Requires X-API-KEY.

POST  https://authcs.vercel.app/api/proxy/user/john_doe/resethwid
Headers: X-API-KEY: YOUR_API_KEY_HERE
Response: { "success":true, "message":"HWID reset successfully", "username":"john_doe" }
DELETE/api/proxy/user/{username}/delete

Permanently deletes a user. This is irreversible. Requires X-API-KEY.

DELETE https://authcs.vercel.app/api/proxy/user/john_doe/delete
Headers: X-API-KEY: YOUR_API_KEY_HERE
Response: { "success":true, "message":"User deleted successfully" }
